# 🔍 VerifyAI — AI-Powered Fake News Detector

> **VerifyAI** is a free, AI-powered web application that detects fake news, misinformation, and misleading content in seconds. Paste any article, headline, or social media post and get an instant AI verdict with detailed credibility scores, bias detection, and actionable tips.

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| **Frontend** | HTML5, CSS3, Vanilla JavaScript |
| **AI Model** | Google Gemini 2.0 Flash Lite |
| **Backend** | Netlify Serverless Functions (Node.js 18) |
| **Charts** | Chart.js 4.4 |
| **Fonts** | Plus Jakarta Sans + JetBrains Mono |
| **Hosting** | Netlify (Free) |

## ⚡ How to Deploy (Free — 5 minutes)

### 1. Get your free Gemini API Key
1. Go to **aistudio.google.com/app/apikey**
2. Sign in with Google → click **"Create API key"**
3. Copy the key (starts with `AIzaSy...`)

### 2. Deploy to Netlify
1. Go to **netlify.com** → sign up free
2. Click **"Add new site"** → **"Deploy manually"**
3. Drag and drop the `FINAL_APP` folder → live in 30 seconds

### 3. Add API Key
1. Netlify → your site → **"Site configuration"**
2. Left sidebar → **"Environment variables"** → **"Add a variable"**
3. Key: `GEMINI_API_KEY` — Value: your Gemini key → **"Create variable"**

### 4. Redeploy
1. Click **"Deploys"** tab
2. Drag the `FINAL_APP` folder again
3. Wait 30 seconds → site is fully live ✅
